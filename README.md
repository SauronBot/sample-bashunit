# sample-bashunit

A sample Bash project using [bashunit](https://github.com/TypedDevs/bashunit) for testing.

## Structure

```
src/
  calculator.sh   # Simple calculator functions
tests/
  calculator_test.sh  # Tests for the calculator
lib/
  bashunit        # Test runner (auto-installed)
```

## Run tests

```bash
./lib/bashunit tests/
```
